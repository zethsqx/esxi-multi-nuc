# esxi-multi-nuc

## Overview

Stretched VLAN across multiple Intel NUC. 
Openshift deploy across a home based setup with minimal available hardware 

## Installation

Items
- Intel NUC
- Mikrotik Switch RB3011UiAS on RouterOS v6.46.3 (stable)
- ESXi 6.7
- OpenShift 4x
